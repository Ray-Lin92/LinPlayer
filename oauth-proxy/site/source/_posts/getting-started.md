几步开始用 LinPlayer 播放你的 Emby 媒体库。

<!-- more -->

## 1. 连接服务器

1. 打开 App，进入「添加服务器」。
2. 填入 Emby 服务器地址、用户名与密码登录。

## 2. 浏览与播放

- 在媒体库中选择影片/剧集，点击播放。
- 支持从上次进度续播。

## 3. 选择内核与字幕

- 播放页「更多 → 内核切换」可在 libmpv / ExoPlayer 间切换。
- 「字幕」可选择内封/外挂字幕、调整延迟；ASS 特效字幕建议用 libmpv 或开启 ExoPlayer 的 libass。

## 4. 手势速查

| 操作 | 手势 |
|---|---|
| 播放/暂停 | 中央点击 |
| 快进/快退 | 双击屏幕右/左 |
| 进度 | 左右滑动 |
| 亮度/音量 | 屏幕左/右上下滑 |
| 临时倍速 | 长按 |

## 5. 第三方同步（可选）

如需 Trakt / Bangumi 同步，相关令牌交换由独立的 OAuth 代理处理（见仓库 `oauth-proxy/`）。
