**LinPlayer** 是一个第三方 Emby 客户端多端播放器（移动 / 桌面 / TV）。

- 源码与 Issue：<https://github.com/zzzwannasleep/LinPlayer>
- 本站为 LinPlayer 的文档与 Wiki，使用 [Hexo](https://hexo.io/) + [Fluid](https://hexo.fluid-dev.com/) 构建，托管于 GitHub Pages。

> 本项目与 Emby 官方无关联，"Emby" 为其各自所有者的商标。
